# تطبيق فيديو قصير (MiaodaShorts)

تطبيق ويب عصري مشابه لتيك توك يتيح للمستخدمين تصفح الفيديوهات القصيرة، تسجيل ونشر محتوى الفيديو، والتفاعل من خلال التعليقات والإعجابات.

## ✨ المميزات

- 📱 **تصفح الفيديوهات**: واجهة تصفح سلسة مع تشغيل تلقائي للفيديوهات
- 🎥 **رفع الفيديوهات**: إمكانية رفع ونشر الفيديوهات من الجهاز
- 💬 **التفاعل**: نظام إعجابات وتعليقات متكامل
- 👥 **المتابعة**: متابعة المستخدمين الآخرين
- 🔍 **البحث**: البحث عن الفيديوهات والمستخدمين
- 👤 **الملف الشخصي**: صفحة ملف شخصي شاملة لكل مستخدم
- 🌙 **الوضع المظلم**: تصميم عصري بوضع مظلم افتراضي
- 📱 **متجاوب**: يعمل بسلاسة على جميع الأجهزة

## 🛠️ التقنيات المستخدمة

- **Frontend**: React + TypeScript + Vite
- **UI Framework**: shadcn/ui + Tailwind CSS
- **Backend**: Firebase (Database + Storage + Auth)
- **Routing**: React Router
- **State Management**: React Context + Hooks

## 🚀 البدء السريع

### المتطلبات

- Node.js 18+ 
- pnpm (أو npm/yarn)

### التثبيت

1. استنساخ المشروع:
```bash
git clone <repository-url>
cd app-a6j4ohtlmv41
```

2. تثبيت الحزم:
```bash
pnpm install
```

3. إعداد متغيرات البيئة:
```bash
# يتم توفير ملف .env تلقائياً مع بيانات Firebase
```

4. تشغيل التطبيق:
```bash
pnpm dev
```

5. فتح المتصفح على: `http://localhost:5173`
